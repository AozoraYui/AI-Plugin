# AI 自画像参考图目录

把 AI 本人的官方/标准形象参考图放在这里，命名为 `0.png`、`1.png`、`2.png` 等（数字越小越靠前）。

当用户在对话里要求“画你自己 / 画一下你 / 给我看看你长什么样”时，画图工具（draw_image）会自动加载本目录下的图片作为参考图喂给绘图模型，用来锁定 AI 的形象，避免模型凭空乱画。

支持的格式：`.png` `.jpg` `.jpeg` `.webp` `.gif`。
最多取前 N 张（受 `MAX_IMAGES_PER_MESSAGE` 配置限制），按文件名排序。

留空（目录里没有图片）时，工具会退回到用 `ai_prompt.yaml` 里的 `self_portrait` 文字描述来画。
